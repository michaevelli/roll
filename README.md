## Roll
Dice roller batch file for Windows

## Usage
<pre>
$ <b>roll 1 2 3</b>
1d2+3
[2]
5
</pre>

## Installation

Put roll.bat into a folder included in the windows Path  
Use command prompt with the command `roll <NumberOfDice> <MaxNumberOnDice> <Modifier>`
